{# 对应 Java index.ftl：stp.isLogin() #}

{# 对应 <@sa.login> / <@sa.notLogin> #}

登录之后才能显示：{% if sa_login() %}value{% endif %}

不登录才能显示：{% if sa_not_login() %}value{% endif %}

{# 对应 <@sa.hasRole value="admin"> 等 #}

具有角色 admin 才能显示：{% if sa_has_role(value="admin") %}value{% endif %}

同时具备多个角色才能显示：{% if sa_has_role_and(value="admin, ceo, cto") %}value{% endif %}

只要具有其中一个角色就能显示：{% if sa_has_role_or(value="admin, ceo, cto") %}value{% endif %}

不具有角色 admin 才能显示：{% if sa_not_role(value="admin") %}value{% endif %}

具有权限 user-add 才能显示：{% if sa_has_permission(value="user-add") %}value{% endif %}

同时具备多个权限才能显示：{% if sa_has_permission_and(value="user-add, user-delete, user-get") %}value{% endif %}

只要具有其中一个权限就能显示：{% if sa_has_permission_or(value="user-add, user-delete, user-get") %}value{% endif %}

不具有权限 user-add 才能显示：{% if sa_not_permission(value="user-add") %}value{% endif %}

从 SaSession 中取值： {% if is_login and session_name %} {{ session_name }} {% endif %}
